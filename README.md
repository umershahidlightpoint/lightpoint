# finance
for accounting / finance POC


## Server app Setup
<code>
cd serverapp <br />
npm install <br />
npx sequelize db:drop<br />
npx sequelize db:create<br />
npx sequelize db:migrate<br />
npx sequelize db:seed:all<br />
npm demo.js <br />
npm run dev<br />
</code>
## Angular App Setup
<code>
npm install <br />
npm start
</code>
